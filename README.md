# LandingPage

## Run by Docker Wordpress

Install Docker && docker compose
https://docs.docker.com/get-docker/

```sh
cd docker
docker compose up
```

## Run by NodeJs (Express.js, Pug.js, stylus.js)
```sh
cd node
npm i
node index.js
```

Open Browser http://localhost:3300
